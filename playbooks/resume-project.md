# Resume Project

Use this playbook when returning to a project after a pause or handoff.

## Steps

1. Read `session-handoff.md`.
2. Read `STATUS.md`.
3. Review active and blocked items in `backlog.md`.
4. Check recent entries in `execution-log.md`.
5. Review recent decisions in `decisions.md`.
6. Confirm the next action before changing files.
7. Update `STATUS.md` if the actual state differs from the written state.

## Resume Checklist

- Current state is understood.
- Next action is explicit.
- Blockers are known.
- Validation expectations are known.
- No hidden assumptions are required to continue.
