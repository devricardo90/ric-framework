# Start New Project

Use this playbook when starting a project with RIC Framework or Protocolo RIC.

## Steps

1. Create the project folder.
2. Copy the templates from `templates/`.
3. Fill in `PROJECT_CONTEXT.md`.
4. Create initial backlog items in `backlog.md`.
5. Set `STATUS.md` to the first active state.
6. Record initial decisions in `decisions.md`.
7. Define validation expectations before implementation starts.
8. Assign roles if agents will be used.

## Minimum Ready State

- Project purpose is clear.
- Scope and exclusions are written.
- First backlog item has acceptance criteria.
- Status has an owner and next action.
- Known risks are visible.
